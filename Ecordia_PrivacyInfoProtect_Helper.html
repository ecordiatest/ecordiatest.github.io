Ecordia Privacy Info Protect Helper

Menu function

File > Open Open a video file to be dealt.
File > Compose Select the target directory for the target video, and then compose the target video file(xxx_gen.mp4).
Firstly, Ensure all rectangle info(position and size for per frame of the video) that need to be protected.
Edit > PreFrame Switch to previous frame, and recognize person and car area via AI.
Edit > NextFrame Switch to next frame, and recognize person and car area via AI.
Edit > AutoNextFrame Switch to next frame automatic and continuous, and recognize person、face and car area via AI.
Edit > AddPOI Add a poi rectangle in the upper left corner of the screen.
Edit > RemovePOI Remove current selected poi rectangle.
Edit > GenerateInPrePostFrame (Reserved)Switch to the next frame, and recognize the brand of car area via find white rectangle for previous frame and next frame by current frame info.
Edit > GenerateInNextFrame (Reserved)Switch to the next frame, and recognize the brand of car area via find white rectangle for next frame by current frame info.
Edit > GenerateInPreFrame (Reserved)Switch to the previous frame, and recognize the brand of car area via find white rectangle for previous frame by current frame info.

Shortcut function

Switch to special frame Switch to special frame via spinbox in statusbar, and recognize person and car area via AI.
Add POI at current position Hold CTRL key, and click left key of mouse.
Remove current selected POI Press DEL key, current poi rectangle is border-color: cyan; border-width: 2px; border-style: dashed;.
Move current selected POI Drag current rectangle.
Resize current selected POI
  • Scroll middle key of mouse to UP, make it bigger.
  • Scroll middle key of mouse to Down, make it smaller.
  • Hold SHIFT key, scroll middle key of mouse, resize it per 10 pixels.
  • Hold w key, scroll middle key of mouse, resize it only for width.
  • Hold h key, scroll middle key of mouse, resize it only for height.
  • Hold SHIFT key, at the same time, hold h key, resize it only for height per 10 pixels.

Rectangle border style

Current selected POI border-color: cyan; border-width: 2px; border-style: dashed;.
POI type: Car/Truck/Motorcycle/Bus border-color: brown; border-style: solid;.
POI type: Person/Face border-color: aquamarine; border-style: solid;.
POI type: yellow/white brand border-color: red; border-style: solid;.
POI type: follow post frame
only for manual mark		 border-color: green; border-style: solid;.
POI type: follow previous frame
only for manual mark		 border-color: yellow; border-style: solid;.
POI type: manual mark border-color: blue; border-style: solid;.
